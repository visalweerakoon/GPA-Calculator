# GPA Calculator

To calculate GPA for multiple semesters. Users can add semesters, add modules with grades and credits, and see both semester wise and cumulative GPA.  

---

## Features

- Add new semesters dynamically.
- Add modules (subjects) with:
  - Module name
  - Grade (A+, A, B-, etc.)
  - Credit hours
- Calculate **semester GPA** and **cumulative GPA**.
- Editable module grades and credits.
- Simple and clean UI with a purple theme.
